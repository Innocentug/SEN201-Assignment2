## Project Overview
This project is a console-based Student Result Management System developed using Python.
The system is designed to help manage student academic records by allowing users to enter student names and scores, automatically calculate grades, and display student results.

## Software Development Life Cycle (SDLC)
The development of this project followed the Software Development Life Cycle (SDLC), which includes:
1. Requirement Analysis
2. System Design
3. Implementation
4. Testing
5. Deployment
6. Maintenance

## Project Features
- Add student name and score
- Automatic grade calculation
- Display student results

## Project Files
- student_result_management_system.py – Main system implementation
- SDLC_Explanation.md – Detailed explanation of the SDLC stages

## How to Run the Project
1. Install Python on your system
2. Open a terminal or command prompt
3. Navigate to the project directory
4. Run the program using:
   python student_result_management_system.py
